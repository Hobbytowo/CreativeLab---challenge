## Description
Single page PSD to HTML recruitment project website

## Built with
- HTML5
- SASS + BEM
- JavaScript
- Gulp

## Setup

``` bash
# install dependencies
$ npm install

# serve with reload
$ gulp watch

# images compression
$ gulp images

# build
$ gulp
```
